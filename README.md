🎵 MusicLotm v1.1.0
MusicLotm is a high-performance, offline-first music player for Android, designed with a focus on minimalist aesthetics and robust state management. Originally built with GetX, the project has been architecturally refactored to Riverpod to ensure scalability and predictable state handling.

✨ What’s New in v1.1.0
Architectural Migration: Fully transitioned from GetX to Riverpod for cleaner dependency injection and state management.

Neumorphic UI Design: A fresh, modern interface leveraging soft shadows and minimalist components for a premium tactile feel.

Advanced Sleep Timer: A custom-engineered timer with real-time countdown and automated playback termination to preserve device battery.

Circular Audio Visualizer: Integrated real-time frequency analysis via the audify package for a dynamic listening experience.

🚀 Key Features
Offline Excellence: Seamlessly scans and manages local READ_MEDIA_AUDIO files without requiring an internet connection.

Background Playback: Rock-solid background audio service using just_audio and audio_service, ensuring music never stops when the screen is locked.

Privacy-First: Zero third-party analytics, zero tracking, and zero ads. All app preferences and metadata are stored 100% locally using Hive.

Dynamic Theming: Responsive Dark Mode and Light Mode support.

🛠️ Tech Stack
Framework: Flutter

State Management: Riverpod

Local Database: Hive (for ultra-fast local storage)

Audio Engine: Just Audio

UI/UX: Neumorphic design principles, custom animations, and flutter_screenutil for responsive scaling.
